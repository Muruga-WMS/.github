# MurugaWMS

MurugaWMS is a comprehensive, microservices-based Warehouse Management System designed for modern, highly automated warehouse environments. The platform leverages Agentic AI and Artificial Intelligence to perform autonomous operational tasks and augment warehouse operations by acting as an intelligent digital co-worker. These AI-driven capabilities support functions such as slotting optimization, wave planning, inventory orchestration, and real-time operational decision support.

## Overview

MurugaWMS is an enterprise-grade warehouse management platform that provides end-to-end visibility, orchestration, and control over warehouse operations. Built on a cloud-native, microservices architecture, the system delivers high scalability, flexibility, resilience, and real-time operational insights.

The platform integrates seamlessly with physical warehouse automation systems such as:
* Automated Storage and Retrieval Systems (ASRS)
* Warehouse Execution Systems (WES)
* Warehouse Control Systems (WCS)
* Goods-to-Person (GTP) systems

MurugaWMS acts as the central orchestration layer that coordinates warehouse workflows, automation systems, enterprise integrations, and operational users.
